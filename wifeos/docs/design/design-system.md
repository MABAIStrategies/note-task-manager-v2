# WifeOS Design System

The accepted direction is a high-fidelity, pink-forward popup-card interface.

## Tokens

- Background: soft cream and blush.
- Primary: rose.
- Accent: mauve.
- Status: sage and terracotta.
- Text: warm charcoal.

## Components

- Popup cards for primary content.
- Swipeable card stacks for secondary modules.
- Bottom sheets and modal cards for creation workflows.
- Compact bottom navigation.
- Sparse animated doodles as secondary accents.

## Image Model Note

Use the newest high-fidelity image generation path available in the working environment. The project records this as `Images 2 or latest high-fidelity image model when available`, with official OpenAI model naming to be rechecked at implementation time.

Generated concept PNGs copied into this folder preserve the initial visual direction.
