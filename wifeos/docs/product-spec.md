# WifeOS Product Spec

WifeOS is a cozy AI command center for baby care, house planning, content creation, watch companionship, and rabbit-hole research.

The MVP is mock-first and focuses on a working demo: useful local state, structured agent outputs, pink popup-card UI, and clear future integration seams.

## MVP Pages

- Today: baby focus, house reset, supplies, Ring, Gmail, Calendar, evening briefing.
- Baby: timeline, add event form, nap prediction, memory capture placeholder.
- House: reset generator, room tasks, laundry, supplies.
- Create: moment input, hooks, captions, baby book entry, monetization ideas, save flow.
- Watch: mood picker, true-crime caseboard, clues, suspects, timeline, prediction.
- Rabbit Holes: topic input, claims, evidence, vibes, scores, verdict.
- Settings: profile, privacy, notifications, Gmail/Calendar/Ring/Alexa placeholders, future multimodal settings.
