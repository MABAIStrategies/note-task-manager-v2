# Privacy Model

MVP default: private mode on.

- Show summaries, not raw Gmail or Calendar payloads.
- Do not send notifications for low-priority context in private mode.
- Keep Ring events quiet unless meaningful.
- Require explicit consent before real Gmail, Calendar, Ring, Alexa, AI, image generation, or image editing integrations.
- Future multimodal workflows must define storage, deletion, and sharing controls before launch.
